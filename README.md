# 🐛🦊🐙Pokedex🐱🦇🐢

This project makes use of the pokeapi.co, the RESTful Pokemon API.
The Pokedex fetches data from the API and presents it on the page.

All the custom scripts are in the "main.js" file in the "js" directory.
